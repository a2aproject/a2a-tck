# CLAUDE.md

## Project Guide

Read `AGENTS.md` at the start of every session for project architecture, conventions, and available commands.

## Skills

At the start of every session, you MUST:

1. Read the **Skills** section of `AGENTS.md` to get the list of available skills.
2. Read every skill's `SKILL.md` file to load its instructions into context.
3. Tell the user which skills have been loaded and briefly describe each one.

When a task matches a loaded skill, follow its `SKILL.md` instructions before acting.
